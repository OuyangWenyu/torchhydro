# Welcome to torchhydro

[![image](https://img.shields.io/pypi/v/torchhydro.svg)](https://pypi.python.org/pypi/torchhydro)

**Deep Learning Toolkit for Hydrology and Water Resources Based on PyTorch**

torchhydro is a specialized deep learning toolkit designed for hydrology and water resources research, offering:

- Pre-processed hydrological datasets
- Specialized data samplers
- Hydrological data transformations
- Pre-trained models
- Domain-specific evaluation metrics

## Key Features

- Rich hydrological dataset support
- Flexible data preprocessing tools
- Multiple hydrological model architectures
- User-friendly API interface
- Comprehensive documentation and examples
- Active community support
